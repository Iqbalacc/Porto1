Tentang Saya

Saya adalah seorang Web Development Enthusiast yang sedang mengembangkan keterampilan di bidang pengembangan web modern. Saya memiliki ketertarikan besar terhadap teknologi web dan saat ini fokus mempelajari HTML, CSS, JavaScript, serta mulai menjajaki framework seperti Tailwind CSS dan React. Sebagai bagian dari perjalanan belajar saya, saya telah membuat beberapa proyek kecil seperti kalkulator berbasis web dan aplikasi To-Do List dengan antarmuka grafis menggunakan Python. Saya memiliki semangat belajar yang tinggi, dan terus berupaya memperluas pengetahuan serta keterampilan untuk dapat berkontribusi dalam pengembangan solusi digital yang fungsional dan user-friendly.
